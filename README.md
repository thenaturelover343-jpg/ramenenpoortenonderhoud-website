# ramenenpoortenonderhoud-website

Basisrepository voor de nieuwe website van Ramen en Poorten Onderhoud.

## Doel
Professionele leadgerichte website opbouwen met duidelijke structuur, sterke lokale vindbaarheid en copy-pasteklare contentflow.

## Voorziene structuur
- `content/` voor websiteteksten in markdown
- `assets/` voor beelden, logo’s en visuele bestanden
- `references/` voor inspiratie en research
- `todo.md` voor concrete volgende stappen

## Eerste focus
1. Positionering scherp zetten
2. Paginastructuur bepalen
3. Kernteksten uitschrijven
4. Designrichting kiezen
5. Bouw en publicatie voorbereiden
